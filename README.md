# shadowizard


#install

`npm i shadowizard --save`

then...


```

import {shadowizard} from 'shadowizard';

shadowizard({
        shadow_type: 'soft',
        padding: false
});


```


## options

optionals :

*   *shadow_type* - _hard | soft_ (default is soft)
*   *padding* - _boolean_ (default is false)
